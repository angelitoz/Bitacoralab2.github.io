¿Que es CSS?

Es un sitio donde se busca darle el estilo único al html. Dándole color y vida al proyecto en sí.¿Cómo interpreta un navegador la información de un archivo html?, la declaración indica el estilo que se aplica en un sector de referencia , puedes escribir CSS en archivos separados o en línea dentro de etiquetas HTML usando el atributo "style". El editor te ayuda con resaltado de sintaxis y sugerencias, y puedes usar extensiones para una experiencia más avanzada, como preprocesadores CSS o herramientas de autocompletado. Recuerda vincular tus archivos CSS a tus páginas HTML para que los estilos se apliquen correctamente.


En la clase la profesora nos enseñó cómo agregar imágenes a nuestras páginas web utilizando HTML y cómo aplicar estilos CSS para controlar su presentación. Comenzamos con la etiqueta y aprendimos a configurar atributos como "src" para especificar la ubicación de la imagen .Luego, nos adentramos en CSS, donde la profesora destacó la propiedad "background-image" para establecer imágenes de fondo en elementos HTML. Aprendimos a usar la propiedad "background-color" para definir el color de fondo de elementos y cómo combinar imágenes y colores de fondo de manera efectiva.

En las clases también nos reunimos por grupos al final de las clases donde entre todos coordinamos para unificar un solo producto , empezamos a entablar y compartir nuestros trabajos de Laboratorio 1 . Propusimos varias ideas buenas y al final de la clase habíamos elegido una , la cual se le comentó a la profesora y con algunas mejoras en lo que serían las modificaciones finales .
